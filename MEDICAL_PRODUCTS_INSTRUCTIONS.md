# كيفية إضافة المنتجات الطبية إلى قاعدة البيانات

تم إنشاء 30 منتجًا طبيًا مع 3 صور لكل منتج. يمكنك إضافتها يدويًا إلى قاعدة بيانات MongoDB الخاصة بك باستخدام إحدى الطريقتين التاليتين:

## الطريقة 1: استخدام ملف JavaScript (للمطورين)

1. تم إنشاء ملف `medical-products-insert.js` يحتوي على أوامر لإدراج كل منتج بشكل فردي.
2. افتح MongoDB Shell أو أي أداة تستخدمها للتفاعل مع قاعدة بيانات MongoDB.
3. نسخ ولصق كل أمر `db.products.insertOne(...)` من الملف في الـ Shell.
4. قم بتنفيذ الأوامر واحدة تلو الأخرى أو جميعها دفعة واحدة.

## الطريقة 2: استخدام ملف JSON (للاستيراد الجماعي)

1. تم إنشاء ملف `medical-products.json` يحتوي على جميع المنتجات بتنسيق JSON.
2. يمكنك استخدام الأمر `mongoimport` لاستيراد جميع المنتجات دفعة واحدة:

```bash
mongoimport --db your_database_name --collection products --file medical-products.json --jsonArray
```

## معلومات إضافية

- جميع المنتجات تنتمي إلى الفئة "medical"
- كل منتج يحتوي على:
  - اسم باللغتين الإنجليزية والعربية
  - وصف باللغتين الإنجليزية والعربية
  - سعر وسعر أصلي
  - مخزون
  - 3 صور لكل منتج
  - ميزات باللغتين الإنجليزية والعربية
  - تصنيف
- بعض المنتجات مميزة بعلامات مثل "جديد"، "مميز"، أو "شائع"

## ملاحظات مهمة

- قبل إضافة هذه المنتجات، تأكد من أن لديك مجموعة "products" في قاعدة بياناتك.
- إذا كانت لديك بيانات موجودة، فكر في إنشاء نسخة احتياطية قبل إضافة منتجات جديدة.
- قد تحتاج إلى تعديل أسماء الحقول إذا كانت تختلف في قاعدة بياناتك الحالية.
